What is CSS?

  1. Css stnads for Cascading Style Sheet
  2. Css describes how HTML elements are to be displayed on screen, paper, or in other media.
  3. Css is not neccesary.Css saves a lot of work. It can control the layout of multiple web pages all once
  4. External stylesheet are stored in CSS files.

    5. CSS can be added in HTML documents

  • Inline- by using the style attribute inside HTML elements1
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS files. However,in this tutorial we will use inline and internal styles, because this is easier to demonstrate , and easier for you to try it yourself.
When comparing CSS class vs ID, the difference is that CSS class applied a style to multiple elements. ID, on other hand, applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS,Selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the commonly used are class and ID. Both are used to target elements to which a style should be applied.
